# 这是什么？

---

一款适用于 Xiaomi HyperOS 的 LSPosed 模块。

适配版本：理论通用

*测试设备为`Pudding`，OS版本`3.0.304.0`，小米澎湃AI引擎版本`3.16.0`*

*（其他设备理论兼容，请自行测试）*

### 核心功能
解除小米“复制直达”强制绑定官方浏览器的限制。

将复制网址弹出的跳转通知，默认调用的浏览器替换为您指定的浏览器，并**同步动态替换浏览器图标**。

### 如何使用

1. 安装并启用该模块。
2. 在 LSPosed 管理器中，勾选作用域 **小米澎湃AI引擎** (`com.xiaomi.aicr`)。
3. 使用MT管理器在`/data/local/tmp/`目录下新建`browser.txt`文件。
4. 在`browser.txt`中，写入你想要替换的浏览器包名（如`com.microsoft.emmx`），并确保文件权限为所有用户可读（`644`）。
5. 重启 **小米澎湃AI引擎**。
6. 享受！
